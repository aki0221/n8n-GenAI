### 1. フィッシングメール対応自動化ワークフロー

```
[メールトリガー] → [メール解析] → [URL/添付ファイル抽出] → [脅威インテリジェンス照会] → [危険度判定] → [条件分岐]
                                                                                      ↓
                                                                    [危険] → [メール隔離] → [ユーザー通知]
                                                                      ↓
                                                                    [安全] → [処理終了]
```

### 2. コンプライアンスチェック自動化ワークフロー

```
[スケジュールトリガー] → [システム設定収集] → [ポリシールール照合] → [違反検出] → [条件分岐]
                                                                    ↓
                                                  [違反あり] → [レポート生成] → [担当者通知]
                                                    ↓
                                                  [違反なし] → [コンプライアンス記録更新]
```
