# パラメータ間の相互関係ネットワーク図の構造設計

## 基本構造

コンセンサスモデルにおけるパラメータ間の相互関係を視覚化するネットワーク図の基本構造を以下のように設計します。

### 1. 全体レイアウト
- **方向**: 上から下（TB）を基本とし、階層構造を明確に表現
- **グループ化**: 6つの主要パラメータグループをサブグラフとして表現
- **中心性**: メタパラメータを中央に配置し、他のパラメータグループとの関係性を放射状に表現

### 2. パラメータグループの配置
```
+---------------------+     +---------------------+
|   評価軸パラメータ   |     |  重み付けパラメータ  |
+---------------------+     +---------------------+
          ↓ ↑                       ↓ ↑
          ↓ ↑                       ↓ ↑
+---------------------+     +---------------------+
|    メタパラメータ    | <-> |  重み付け方法パラメータ |
+---------------------+     +---------------------+
          ↑ ↓                       ↑ ↓
          ↑ ↓                       ↑ ↓
+---------------------+     +---------------------+
|    閾値パラメータ    |     |    調整パラメータ    |
+---------------------+     +---------------------+
```

## 詳細設計

### 1. 評価軸パラメータ（青系）
- **形状**: 四角形 `[]`
- **サブグループ**:
  - 視点レベルパラメータ（濃い青）
  - 視点内評価要素（薄い青、3つのサブグループ）

```
subgraph 評価軸パラメータ
    subgraph 視点レベルパラメータ
        テクノロジー視点の評価スコア
        マーケット視点の評価スコア
        ビジネス視点の評価スコア
    end
    
    subgraph テクノロジー視点評価要素
        技術成熟度
        実用化可能性
        技術的優位性
    end
    
    subgraph マーケット視点評価要素
        市場成長性
        競合状況
        顧客需要
    end
    
    subgraph ビジネス視点評価要素
        収益性
        戦略的適合性
        リスク
    end
end
```

### 2. 重み付けパラメータ（緑系）
- **形状**: 丸角四角形 `()`
- **サブグループ**:
  - 視点レベルの重み係数（濃い緑）
  - 評価要素レベルの重み係数（薄い緑、3つのサブグループ）

```
subgraph 重み付けパラメータ
    subgraph 視点レベルの重み係数
        テクノロジー視点の重み
        マーケット視点の重み
        ビジネス視点の重み
    end
    
    subgraph テクノロジー視点要素の重み係数
        技術成熟度の重み
        実用化可能性の重み
        技術的優位性の重み
    end
    
    subgraph マーケット視点要素の重み係数
        市場成長性の重み
        競合状況の重み
        顧客需要の重み
    end
    
    subgraph ビジネス視点要素の重み係数
        収益性の重み
        戦略的適合性の重み
        リスクの重み
    end
end
```

### 3. 重み付け方法パラメータ（黄系）
- **形状**: 六角形 `{{}}`
- **サブグループ**:
  - 重み付け方法の種類
  - 動的重み付け調整要因

```
subgraph 重み付け方法パラメータ
    subgraph 重み付け方法の種類
        静的重み付け
        動的重み付け
        コンテキスト依存重み付け
    end
    
    subgraph 動的重み付け調整要因
        トピックの性質
        変化の段階
        各視点の情報の確信度
    end
end
```

### 4. 閾値パラメータ（赤系）
- **形状**: 菱形 `{}`
- **サブグループ**:
  - 評価レベル閾値
  - 信頼性閾値

```
subgraph 閾値パラメータ
    subgraph 評価レベル閾値
        高評価の閾値
        中評価の閾値
        低評価の閾値
    end
    
    subgraph 信頼性閾値
        最小信頼度閾値
        再評価トリガー閾値
    end
end
```

### 5. 調整パラメータ（オレンジ系）
- **形状**: 円形 `(())`
- **サブグループ**:
  - 時間的調整因子
  - コンテキスト調整因子

```
subgraph 調整パラメータ
    subgraph 時間的調整因子
        短期影響の重み
        中期影響の重み
        長期影響の重み
    end
    
    subgraph コンテキスト調整因子
        業界特性調整係数
        組織規模調整係数
        地域特性調整係数
    end
end
```

### 6. メタパラメータ（紫系）
- **形状**: 円柱形 `[()]`
- **サブグループ**:
  - モデル制御パラメータ
  - フィードバックパラメータ

```
subgraph メタパラメータ
    subgraph モデル制御パラメータ
        評価サイクル頻度
        重み再調整頻度
        履歴データ参照期間
    end
    
    subgraph フィードバックパラメータ
        予測精度フィードバック係数
        ユーザー満足度フィードバック係数
        意思決定貢献度フィードバック係数
    end
end
```

## 相互関係の表現

### 1. 階層的依存関係（実線矢印 `-->`)
- 視点内評価要素 --> 視点レベルパラメータ
- 視点レベルパラメータ --> 総合評価スコア（メタパラメータ内に表現）

### 2. 影響関係（点線矢印 `-..->`）
- 重み付け方法パラメータ -.-> 重み付けパラメータ
- 閾値パラメータ -.-> 評価軸パラメータ
- 調整パラメータ -.-> 評価軸パラメータ

### 3. 調整関係（太線矢印 `==>`）
- メタパラメータ ==> 重み付け方法パラメータ
- メタパラメータ ==> 閾値パラメータ
- メタパラメータ ==> 調整パラメータ

### 4. フィードバック関係（曲線矢印 `--o`）
- 評価軸パラメータ --o メタパラメータ
- 重み付けパラメータ --o メタパラメータ

## 凡例

```
classDef evaluationParams fill:#d0e0ff,stroke:#0066cc,stroke-width:1px;
classDef weightParams fill:#d0ffe0,stroke:#00cc66,stroke-width:1px;
classDef weightMethodParams fill:#fff8d0,stroke:#ccaa00,stroke-width:1px;
classDef thresholdParams fill:#ffd0d0,stroke:#cc0000,stroke-width:1px;
classDef adjustmentParams fill:#ffe0d0,stroke:#cc6600,stroke-width:1px;
classDef metaParams fill:#e0d0ff,stroke:#6600cc,stroke-width:1px;

class [評価軸パラメータ内のノード] evaluationParams;
class [重み付けパラメータ内のノード] weightParams;
class [重み付け方法パラメータ内のノード] weightMethodParams;
class [閾値パラメータ内のノード] thresholdParams;
class [調整パラメータ内のノード] adjustmentParams;
class [メタパラメータ内のノード] metaParams;
```

## 注釈

- 各パラメータグループの重要性や影響度を視覚的に表現するため、ノードのサイズを調整
- 特に重要な関係性は太い線や目立つ色で強調
- n8nでの実装との対応を示すため、関連するn8nノードタイプを注釈として追加
- 複雑な相互作用を持つパラメータには、詳細な説明を吹き出しとして追加
