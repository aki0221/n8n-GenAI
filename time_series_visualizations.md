# 視覚的要素の追加

## 時系列予測モデルの概念図

```
+---------------------+     +---------------------+     +---------------------+
|                     |     |                     |     |                     |
|  過去のデータパターン  +---->  時系列予測モデル   +---->  将来値の予測       |
|                     |     |                     |     |                     |
+---------------------+     +---------------------+     +---------------------+
                            |                     |
                            |  • ARIMA            |
                            |  • 指数平滑法       |
                            |  • Prophet         |
                            |                     |
                            +---------------------+
```

## モデル選択フローチャート

```
開始
  |
  v
[データ量の評価]
  |
  +----> 少量（<30ポイント） ----> 単純指数平滑法/移動平均法
  |
  +----> 中量（30-100ポイント） ----> ホルト・ウィンタース法/ARIMA
  |
  +----> 大量（>100ポイント） ----> ARIMA/Prophet/機械学習モデル
  |
  v
[季節性の評価]
  |
  +----> 明確な季節性あり ----> ホルト・ウィンタース法/季節性ARIMA/Prophet
  |
  +----> 季節性なし/弱い ----> ARIMA/単純指数平滑法
  |
  v
[トレンドの評価]
  |
  +----> 線形トレンドあり ----> ホルト法/ARIMA/Prophet
  |
  +----> 非線形トレンド ----> Prophet
  |
  +----> トレンドなし/弱い ----> 単純指数平滑法/ARIMA(0,0,q)
  |
  v
[予測期間の考慮]
  |
  +----> 短期（1日-1ヶ月） ----> ARIMA/ホルト・ウィンタース法
  |
  +----> 中期（1-6ヶ月） ----> 季節性ARIMA/Prophet
  |
  +----> 長期（>6ヶ月） ----> Prophet/構造時系列モデル
  |
  v
[外部要因の重要性]
  |
  +----> 外部要因重要 ----> ARIMAX/Prophet+回帰変数/機械学習
  |
  +----> 外部要因不要 ----> ARIMA/指数平滑法/Prophet
  |
  v
[解釈可能性の要件]
  |
  +----> 高い解釈可能性必要 ----> 指数平滑法/ARIMA
  |
  +----> 予測精度優先 ----> Prophet/アンサンブル/機械学習
  |
  v
最適モデル選択完了
```

## 各モデルの特性比較表

| モデル | 強み | 弱み | 適したデータ特性 | 計算負荷 | 実装の複雑さ |
|-------|------|------|--------------|--------|------------|
| 単純指数平滑法 | シンプル、少ないデータでも機能 | トレンドや季節性を扱えない | トレンドや季節性のないデータ | 非常に低い | 低い |
| ホルト法 | トレンドを扱える | 季節性を扱えない | トレンドはあるが季節性のないデータ | 低い | 低い |
| ホルト・ウィンタース法 | トレンドと季節性を扱える | 複雑な季節パターンには不向き | 明確なトレンドと季節性を持つデータ | 中程度 | 中程度 |
| ARIMA | 柔軟性が高い、自己相関を活用 | パラメータ選択が難しい | 定常的または差分で定常化可能なデータ | 中程度 | 高い |
| 季節性ARIMA | 季節性を扱える | パラメータ選択がさらに複雑 | 季節性を持つ時系列データ | 高い | 高い |
| Prophet | 複数の季節性、休日効果、変化点を自動検出 | ブラックボックス的、カスタマイズ性に制限 | 複雑な季節パターン、長期予測 | 非常に高い | 中程度 |
